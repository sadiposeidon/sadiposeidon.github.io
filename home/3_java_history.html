Java'nın Tarihi

Java programlama dili ilk olarak James Gosling tarafından başlatılan ve Sun Microsystems'in Java platformunun (Java 1.0 [J2SE]) temel bileşeni olarak 1995 yılında piyasaya sürülen Sun Microsystems tarafından geliştirildi. Java'ya isim vermenin bile tarihi çok ilginçtir. Birçok ismin altına girdi.

Java Adı Geçmişi

YeşilTalk (GreenTalk)

James Gosling 'Yeşil' takım olarak adlandırılan bir takıma liderlik ediyordu. Bu ekibin hedefi bunu yapabilecek yeni bir proje yaratmaktı. Başlangıçta projeyi geliştirmek için ilk tercih C++ idi. James Gosling hedefe ulaşmak için C++'ı geliştirmek istedi ancak yüksek bellek kullanımı nedeniyle bu fikir reddedildi ve ekip başlangıçta GreenTalk olarak adlandırılan yeni bir dille başladı. .gt olarak kullanılan dosya uzantısı. Daha sonra bu dil Meşe (Oak) ve son olarak da Java olarak adlandırıldı.

Meşe (Oak)

James Gosling dili Oak olarak yeniden adlandırdı. Ofisinin önünde bir meşe ağacı vardı. James Gosling, meşenin dayanışmayı temsil etmesi ve meşe ağacının ABD, Fransa, Romanya gibi birçok ülkenin milli ağacı olması nedeniyle bu ismi kullanmıştır. Ancak Oak teknolojilerinde zaten Oak'ın ticari markası vardı ve James ekibinin dil için başka bir başlık üzerinde beyin fırtınası yapması gerekti.

Sonunda Java

Ekip DNA, Silk, Ruby ve Java gibi birden fazla isim koydu. Java ekip tarafından tamamlandı. James Gosling, Java başlığını espresso kahve çekirdeğinin türüne göre sıraladı. Java, Endonezya'da Java kahvesi adı verilen yeni kahvenin keşfedildiği bir adadır. James Gosling'e göre Java, Silk'le birlikte en çok tercih edilenler arasındaydı. Son olarak Java oldukça benzersiz olduğu ve dinamik, devrimci ve söylemesi eğlenceli olmanın özünü temsil ettiği için seçildi.

Sun, ilk genel uygulamayı 1995 yılında Java 1.0 olarak piyasaya sürdü. Bir Kez Yaz, Her Yerde Çalıştır Write Once, Run Anywhere (WORA) sözü verdi ve popüler platformlarda ücretsiz çalışma süreleri sağladı.

13 Kasım 2006'da Sun, Java'nın çoğunu GNU Genel Kamu Lisansı (GPL) koşulları altında ücretsiz ve açık kaynaklı yazılım olarak yayınladı.

8 Mayıs 2007'de Sun süreci tamamladı ve Sun'ın telif hakkına sahip olmadığı küçük bir kod kısmı dışında Java'nın tüm çekirdek kodlarını ücretsiz ve açık kaynak haline getirdi.

Java Standard Edition'ın en son sürümü Java SE 21'dir. Java'nın ilerlemesi ve yaygınlaşmasıyla birlikte, çeşitli platform türlerine uyacak şekilde birden fazla yapılandırma oluşturuldu. Örneğin: Kurumsal Uygulamalar için J2EE, Mobil Uygulamalar için J2ME.

Java Sürüm Geçmişi

Yaklaşık 30 yıllık bir süre boyunca Java birçok küçük ve büyük sürüm gördü. Aşağıda bugüne kadarki Java sürümlerinin kısa bir açıklaması bulunmaktadır.

1 | JDK Betası | 1995 | İlk Taslak versiyonu

2 | JDK1.0 | 23 Ocak 1996 | JDK 1.0.2'nin kararlı bir çeşidi JDK 1 olarak adlandırıldı

3 | JDK1.1 | 19 Şubat 1997 | Bu sürümde JavaBeans, RMI, JDBC, iç sınıflar gibi önemli özellikler eklendi.

4 | JDK 1.2 | 8 Aralık 1998 | Swing, JIT Derleyicisi, Java Modülleri, Koleksiyonlar JAVA'ya tanıtıldı ve bu sürüm büyük bir başarı elde etti.

5 | JDK 1.3 | 8 Mayıs 2000 | HotSpot JVM, JNDI, JPDA, JavaSound ve Sentetik proxy sınıfları desteği eklendi.

6 | JDK1.4 | 6 Şubat 2002 | JPEG/PNG görüntüsü oluşturmak/okumak için Görüntü G/Ç API'si eklendi. Entegre XML ayrıştırıcı ve XSLT işlemci (JAXP) ve Tercihler API'si diğer önemli güncellemelerdi.

7 | JDK 1.5 veya J2SE 5 | 30 Eylül 2004 | Dile foreach, var-args, generals vb. gibi çeşitli yeni özellikler eklendi.

8 | JAVASE 6 | 11 Aralık 2006 | 1. notasyon SE'ye düşürüldü ve JAXB 2.0'a yükseltmeler yapıldı, JSR 269 desteği ve JDBC 4.0 desteği eklendi.

9 | JAVA SE 7 | 7 Temmuz 2011 | JVM'ye dinamik dil desteği eklendi. Diğer geliştirmeler arasında anahtar durumunda dize, sıkıştırılmış 64 bit işaretçiler vb. yer aldı.

10 | JAVASE 8 | 18 Mart 2014 | Fonksiyonel programlama desteği eklendi. Lambda ifadeleri, akışlar, varsayılan yöntemler, yeni tarih-saat API'leri tanıtıldı.

11 | JAVA SE 9 | 21 Eylül 2017 | JVM platformuna uygulanabilecek modül sistemi tanıtıldı.

12 | JAVA SE 10 | 20 Mart 2018 | Unicode dil etiketi uzantıları eklendi. Kök sertifikalar, yerel iş parçacığı anlaşmaları, alternatif bellek cihazlarında yığın tahsisi desteği vb. tanıtıldı.

13 | JAVA SE 11 | 5 Eylül 2018 | Dinamik sınıf dosyası sabitleri, işlem gerektirmeyen bir çöp toplayıcı olan Epsilon, lambda parametrelerinde yerel değişken desteği, Düşük yük yığın profili oluşturma desteği eklendi.

14 | JAVA SE 12 | 19 Mart 2019 | Deneysel Çöp Toplayıcı, Shenandoah: Düşük Duraklatma Süreli Çöp Toplayıcı, Microbenchmark Suite, JVM Sabitleri API'si eklendi.

15 | JAVA SE 13 | 17 Eylül 2019 | Özellik eklendi - Metin Blokları (Çok satırlı dizeler), Geliştirilmiş Konu-yerel anlaşmalar.

16 | JAVA SE 14 | 17 Mart 2020 | Özellik eklendi - Kayıtlar, modelleme için yeni bir sınıf türü, örneğin için Desen Eşleştirme, Sezgisel NullPointerException işleme.

17 | JAVA SE 15 | 15 Eylül 2020 | Özellik eklendi - Mühürlü Sınıflar, Gizli Sınıflar, Yabancı İşlev ve Bellek API'si (Kuluçka Makinesi).

18 | JAVA SE 16 | 16 Mart 2021 | Önizleme olarak eklenen özellik - Kayıtlar, Anahtar için Desen Eşleştirme, Unix Etki Alanı Soket Kanalı (Kuluçka Makinesi) vb.

19 | JAVA SE 17 | 14 Eylül 2021 | Özellik sonlandırıldıkça eklendi - Mühürlü Sınıflar, örneğin Örneği için Desen Eşleştirme, varsayılan olarak JDK dahili öğelerinin güçlü kapsüllenmesi. Yeni macOS oluşturma hattı vb.

20 | JAVA SE 18 | 22 Mart 2022 | Özellik eklendi - Varsayılan olarak UTF-8, Java API Dokümantasyonunda Kod Parçacıkları, Vektör API (Üçüncü Kuluçka Makinesi), Yabancı İşlev, Bellek API'si (İkinci Kuluçka Makinesi) vb.

21 | JAVA SE 19 | 20 Eylül 2022 | Özellik eklendi - Kayıt modeli, Vektör API (Dördüncü inkübatör), Yapılandırılmış Eşzamanlılık (Inkübatör) vb.

22 | JAVA SE 20 | 21 Mart 2023 | Özellik eklendi - Kapsamlı Değerler (İnkübatör), Kayıt Desenleri (İkinci Önizleme), Anahtar için Desen Eşleştirme (Dördüncü Önizleme), Yabancı Fonksiyon ve Bellek API'si (İkinci Önizleme) vb.

22 | JAVA SE 21 | 19 Eylül 2023 | Özellik eklendi - Dizi Şablonları (Önizleme), Sıralı Koleksiyonlar, Nesil ZGC, Kayıt Modelleri, Anahtar için Desen Eşleştirme vb.
