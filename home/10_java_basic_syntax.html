Java - Temel Sözdizimi (Basic Syntax)

Bir Java programını ele aldığımızda, birbirlerinin yöntemlerini çağırarak iletişim kuran nesnelerin topluluğu olarak tanımlanabilir. Şimdi kısaca sınıf, nesne, yöntem ve örnek değişkenlerin ne anlama geldiğine bakalım.

Nesne (Object) – Nesnelerin durumları ve davranışları vardır. Örnek: Bir köpeğin renk, isim, cins gibi halleri ve ayrıca kuyruğunu sallama, havlama, yemek yeme gibi davranışları vardır. Bir nesne bir sınıfın örneğidir.

Sınıf (Class) – Bir sınıf, kendi türündeki nesnenin desteklediği davranışı/durumu tanımlayan bir şablon/plan olarak tanımlanabilir.

Yöntemler (Methods) - Bir yöntem temel olarak bir davranıştır. Bir sınıf birçok yöntem içerebilir. Mantıkların yazıldığı, verilerin değiştirildiği ve tüm eylemlerin yürütüldüğü yöntemlerdedir.

Örnek Değişkenleri (Instance Variables) – Her nesnenin kendine özgü örnek değişkenleri kümesi vardır. Bir nesnenin durumu, bu örnek değişkenlere atanan değerler tarafından oluşturulur.

Temel Söz Dizimi (Basic Syntax)

Büyük/Küçük Harfe Duyarlılık - Java büyük/küçük harfe duyarlıdır; bu, Merhaba ve merhaba tanımlayıcılarının Java'da farklı anlamlara sahip olacağı anlamına gelir.

Sınıf Adları - Tüm sınıf adlarında ilk harf Büyük Harf olmalıdır. Sınıfın adını oluşturmak için birden fazla kelime kullanılıyorsa, her iç kelimenin ilk harfi Büyük Harf olmalıdır.

Örnek - sınıf (class) MyFirstJavaClass

Yöntem Adları - Tüm yöntem adları Küçük harfle başlamalıdır. Yöntemin adını oluşturmak için birden fazla kelime kullanılıyorsa, her iç kelimenin ilk harfi Büyük Harf olmalıdır.

Örnek – public void myMethodName()

Program Dosyası Adı – Program dosyasının adı sınıf adıyla tam olarak eşleşmelidir.

Dosyayı kaydederken, sınıf adını kullanarak kaydetmeli (Java'nın büyük/küçük harfe duyarlı olduğunu unutmayın) ve adın sonuna .java eklemelisiniz (dosya adı ile sınıf adı eşleşmiyorsa programınız derlenmez).

Ancak, dosyada genel bir sınıfınızın bulunmaması durumunda dosya adının sınıf adından farklı olabileceğini lütfen unutmayın. Ayrıca dosyada public bir sınıfın bulunması zorunlu değildir.

Örnek – 'MyFirstJavaProgram'ın sınıf (class) adı olduğunu varsayalım. Daha sonra dosya MyFirstJavaProgram.java olarak kaydedilmelidir.

public static void main(String args[]) – Java programı işleme, her Java programının zorunlu bir parçası olan main() yönteminden başlar.

Java Tanımlayıcıları (Java Identifiers)

Tüm Java bileşenleri ad gerektirir. Sınıflar, değişkenler ve yöntemler için kullanılan adlara tanımlayıcılar denir . Java'da tanımlayıcılarla ilgili hatırlanması gereken birkaç nokta vardır. Bunlar aşağıdaki gibidir.

Tüm tanımlayıcılar bir harfle (A'dan Z'ye veya a'dan z'ye), para birimi karakteriyle ($) veya alt çizgiyle (_) başlamalıdır.

İlk karakterden sonra tanımlayıcılar herhangi bir karakter kombinasyonuna sahip olabilir.

Anahtar kelime tanımlayıcı olarak kullanılamaz.

En önemlisi, tanımlayıcılar büyük/küçük harfe duyarlıdır.

Yasal tanımlayıcılara örnekler: yaş, $salary, _value, __1_value.

Yasadışı tanımlayıcılara örnekler: 123abc, -salary.

Java Değiştiricileri (Java Modifiers)

Diğer dillerde olduğu gibi değiştiriciler kullanılarak sınıfları, yöntemleri vb. değiştirmek mümkündür. İki değiştirici kategorisi vardır –

Erişim Değiştiricileri (Access Modifiers) - default, public , protected, private

Erişimsiz Değiştiriciler (Non-access Modifiers) - final, abstract, strictfp

Java Dizileri (Arrays)

Diziler aynı türden birden fazla değişkeni saklayan nesnelerdir. Ancak dizinin kendisi yığındaki bir nesnedir. Gelecek bölümlerde nasıl deklare edileceğini, oluşturulacağını ve başlatılacağını inceleyeceğiz.

Java Numaralandırmaları, sıralamaları (Enums)

Numaralandırmalar Java 5.0'da tanıtıldı. Numaralandırmalar, bir değişkeni yalnızca birkaç önceden tanımlanmış değerden birine sahip olacak şekilde kısıtlar. Bu numaralandırılmış listedeki değerlere numaralandırma denir.

Numaralandırmaların kullanımıyla kodunuzdaki hata sayısını azaltmak mümkündür.

Örneğin bir taze meyve suyu mağazası için başvuru yaparsak bardak boyutunu küçük, orta ve büyük olarak sınırlamak mümkün olacaktır. Bu, kimsenin küçük, orta veya büyük beden dışında herhangi bir beden sipariş etmesine izin vermeyeceğinden emin olacaktır. 

class FreshJuice {

   enum FreshJuiceSize{ SMALL, MEDIUM, LARGE }
   FreshJuiceSize size;
}

public class FreshJuiceTest {

   public static void main(String args[]) {

      FreshJuice juice = new FreshJuice();
      juice.size = FreshJuice.FreshJuiceSize.MEDIUM ;

      System.out.println("Size: " + juice.size);
   }
}

Not – Numaralandırmalar kendilerine ait olarak veya bir sınıf içinde bildirilebilir. Yöntemler, değişkenler, yapıcılar numaralandırmaların içinde de tanımlanabilir.

4 break - Java programlama dilinde break ifadesi aşağıdaki iki kullanıma sahiptir. Bir döngü içinde break deyimiyle karşılaşıldığında, döngü hemen sonlandırılır ve program kontrolü, döngüyü takip eden bir sonraki deyimden devam eder. Switch ifadesindeki bir durumu sonlandırmak için kullanılabilir.

5 byte - bayt veri türü, Java tarafından desteklenen sekiz temel veri türünden biridir. Bir bayt değerini kabul edebilen bayt tipi değişkenler oluşturmanın yollarını sağlar.

6 case - case anahtar sözcüğü, bir değişkenin bir değerler listesine göre eşitlik açısından test edilmesine olanak tanıyan switch ifadesinin bir parçasıdır.

7 catch - İstisna (veya istisnai olay), bir programın yürütülmesi sırasında ortaya çıkan bir sorundur.

8 char - char veri türü, Java tarafından desteklenen sekiz temel veri türünden biridir.

9 class - Java Nesneye Yönelik Bir Dildir. Nesneye Dayalı özelliğe sahip bir dil olarak.

10 const - final anahtar sözcüğü, Java'da sabit değeri veya son yöntemleri/sınıfları tanımlamak için kullanılır.

11 continue - Continue anahtar sözcüğü herhangi bir döngü kontrol yapısında kullanılabilir.

12 default - default anahtar sözcüğü, bir değişkenin bir değerler listesine göre eşitlik açısından test edilmesine olanak tanıyan switch ifadesinin bir parçasıdır.

Java'daki yorumlar

Java, C ve C++'a çok benzer şekilde tek satırlı ve çok satırlı yorumları destekler. Herhangi bir yorumun içinde bulunan tüm karakterler Java derleyicisi tarafından göz ardı edilir. 

public class MyFirstJavaProgram {

   /* Bu benim ilk Java programım.
    * Bu, çıktı olarak 'Merhaba Dünya'yı yazdıracaktır
    * Bu çok satırlı yorumların bir örneğidir.
    */

   public static void main(String []args) {
      // Bu tek satırlık yoruma bir örnektir
       /* Bu aynı zamanda tek satırlık yorum örneğidir. */

      System.out.println("Merhaba Dünya");
   }
}

Boş Satırları Kullanma

Muhtemelen bir açıklama içeren, yalnızca beyaz boşluk içeren bir satır boş satır olarak bilinir ve Java bunu tamamen yok sayar.

Miras (Inheritance)

Java'da sınıflar sınıflardan türetilebilir. Temel olarak, yeni bir sınıf oluşturmanız gerekiyorsa ve ihtiyacınız olan kodun bir kısmını içeren bir sınıf zaten varsa, o zaman yeni sınıfınızı mevcut koddan türetmeniz mümkündür.

Bu kavram, yeni bir sınıfta kodu yeniden yazmaya gerek kalmadan mevcut sınıfın alanlarını ve yöntemlerini yeniden kullanmanıza olanak tanır. Bu senaryoda, mevcut sınıfa üst sınıf , türetilmiş sınıfa ise alt sınıf adı verilir .

Arayüzler (Interfaces)

Java dilinde arayüz, nesnelerin birbirleriyle nasıl iletişim kuracağına ilişkin bir sözleşme olarak tanımlanabilir. Miras kavramı söz konusu olduğunda arayüzler hayati bir rol oynamaktadır.

Bir arayüz, türeyen bir sınıfın (alt sınıfın) kullanması gereken yöntemleri tanımlar. Ancak yöntemlerin uygulanması tamamen alt sınıfa bağlıdır.

Bir sonraki bölümde Java programlamadaki Nesneler ve sınıflar anlatılmaktadır. Oturumun sonunda Java'da nesnelerin ve sınıfların ne olduğu konusunda net bir fikir edinebileceksiniz.
