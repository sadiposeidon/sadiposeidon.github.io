Java nedir?

Java, orijinal olarak Sun Microsystems tarafından geliştirilen ve 1995 yılında piyasaya sürülen popüler bir üst düzey, nesne yönelimli programlama (object-oriented programming) dilidir. Şu anda Java, Oracle'a aittir ve 3 milyardan fazla cihaz Java'yı çalıştırmaktadır. Java, Windows , Mac OS ve UNIX'in çeşitli sürümleri gibi çeşitli platformlarda çalışır . Bugün Java, Masaüstü Uygulamaları, Mobil uygulamalar, Web uygulamaları, Oyunlar ve çok daha fazlasını içeren çok sayıda yazılım uygulaması türünü geliştirmek için kullanılıyor.

WORA

Java, programcıların Bir Kez Yaz, Her Yerde Çalıştır Write Once, Run Anywhere (WORA) yapmasına olanak tanıyan genel amaçlı bir programlama dilidir . Bu, derlenmiş Java kodunun, yeniden derlemeye gerek kalmadan Java'yı destekleyen tüm platformlarda çalışabileceği anlamına gelir.

public class MyFirstJavaProgram {

   /* Bu benim ilk Java programım.
    * Bu, çıktı olarak 'Merhaba, Dünya!' yazacaktır.

Java'nın Özellikleri

Java zengin özelliklere sahip bir dildir. Java her güncellemeyle sürekli olarak gelişmektedir ve güncellemeler her altı ayda bir gelmektedir. Aşağıda Java dilinin temel özelliklerinden bazıları verilmiştir.

Nesne yönelimli (Object Oriented): Java saf nesne yönelimli bir dildir ve Java'daki her şey bir nesnedir. Java, Kalıtım (Inheritance), Kapsülleme (Encapsulation), Polimorfizm (Polymorphism), Sınıflar (Classes) vb. gibi OOPS ilkelerini destekler. Java'nın kendisi bir nesne modeline dayandığı gibi genişletilebilir.

Platformdan bağımsız (Platform Independent): Java kodu platformdan bağımsızdır. Java kodu makineye özel kod halinde derlenmez, platformdan bağımsız bir bayt kodu halinde derlenir. Bu bayt kodu , kodu temel platformda çalıştıran JVM tarafından yürütülür . Bu yetenek, Java'yı Bir Kez Yaz, Her Yerde Çalıştır dili haline getirir.

Öğrenmesi kolay (Easy To Learn): Java, C ve C++'ın özelliklerini devralır ve geliştiriciler, C veya C++ dillerinden herhangi birini biliyorlarsa Java'yı kolayca öğrenebilirler. Bilgisayar dillerine yeni başlayan biri için bile Java'yı sıfırdan öğrenmek çok kolaydır.

Güvenli (Secure): Java mimarisi gereği güvenlidir. Bir geliştiricinin temeldeki bellek veya İşletim Sistemiyle doğrudan etkileşimde bulunmasına gerek yoktur. Java otomatik çöp toplama olanağı sağlar, böylece geliştiricilerin bellek sızıntıları, yönetim vb. konularda endişelenmelerine gerek kalmaz.

Mimari-Nötr (Architectural-Neutral): Java bayt kodu her türlü işlemcide çalıştırılabilir. JRE, farklı işlemci türlerinde kod yürütmeyi otomatik olarak gerçekleştirir.

Taşınabilir (Portable): Windows makinesinde yazılan bir Java kodu, MacOS'ta herhangi bir kod değişikliği yapılmadan yürütülebilir ve bunun tersi de geçerlidir. İşletim sistemine özel herhangi bir kod değişikliği yapmanıza gerek yoktur.

Sağlamlık (Robust): Java, çok güçlü derleme zamanı hata kontrolleri, sıkı tür kontrolü ve çalışma zamanı istisna yönetimi ile çok sağlam bir dildir.

Çoklu iş parçacığı (Multithreading): Java, çoklu işlem ve çoklu iş parçacığı için yerleşik destek sağlar. Java iş parçacığı yönetimi, monitörler, kilitlenme yönetimi, yarış koşulları vb. sağlar.

Yüksek performans (High Performance): Java yorumlanmasına rağmen hala çok performanslıdır. JIT (Tam Zamanında) (Just In Time) derleyicisi (compiler) performansı artırmaya yardımcı olur.

Dağıtılmış (Distributed): Java, dağıtılmış sistemler için tasarlanmıştır ve internet dağıtılmış bir ortam olduğundan internet tabanlı uygulamalar geliştirmek için en popüler dildir.
