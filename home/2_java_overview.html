Java - Genel Bakış

Java programlama dili ilk olarak James Gosling tarafından başlatılan ve Sun Microsystems'in Java platformunun (Java 1.0 [J2SE]) temel bileşeni olarak 1995 yılında piyasaya sürülen Sun Microsystems tarafından geliştirildi.

Java Standard Edition'ın en son sürümü Java SE 8'dir. Java'nın gelişmesi ve yaygınlaşmasıyla birlikte, çeşitli platform türlerine uyacak şekilde birden fazla yapılandırma oluşturuldu. Örneğin: Kurumsal Uygulamalar için J2EE, Mobil Uygulamalar için J2ME.

Yeni J2 sürümleri sırasıyla Java SE, Java EE ve Java ME olarak yeniden adlandırıldı. Java'nın Bir Kez Yaz, Her Yerde Çalıştır Write Once, Run Anywhere (WORA) özelliği garanti edilir.

Nesneye yönelik (Object Oriented): Java'da her şey bir Nesnedir. Java, Object modelini temel aldığından kolaylıkla genişletilebilir.

Platformdan bağımsız (Platform Independent): C ve C++ da dahil olmak üzere diğer birçok programlama dilinden farklı olarak Java derlendiğinde, platforma özel bir makinede değil, platformdan bağımsız bayt kodunda derlenir. Bu bayt kodu web üzerinden dağıtılır ve hangi platformda çalıştırılırsa çalıştırılsın Sanal Makine (JVM) tarafından yorumlanır.

Basit (Simple): Java öğrenmesi kolay olacak şekilde tasarlanmıştır. OOP Java'nın temel konseptini anlarsanız, ustalaşması kolay olacaktır.

Güvenli (Secure): Java'nın güvenli özelliği sayesinde virüssüz, kurcalanmaya karşı korumalı sistemler geliştirmenize olanak sağlar. Kimlik doğrulama teknikleri genel anahtar şifrelemesine dayanmaktadır.

Mimariden bağımsız (Architecture-neutral): Java derleyicisi, Java çalışma zamanı sisteminin varlığıyla derlenmiş kodu birçok işlemcide yürütülebilir hale getiren, mimariden bağımsız bir nesne dosyası formatı oluşturur.

Taşınabilir (Portable): Mimariden bağımsız olması ve spesifikasyonun uygulamaya bağımlı yönlerinin bulunmaması, Java'yı taşınabilir kılar. Java'daki derleyici, bir POSIX alt kümesi olan temiz taşınabilirlik sınırıyla ANSI C'de yazılmıştır.

Sağlamlık (Robust): Java, esas olarak derleme zamanı hata denetimi ve çalışma zamanı denetimi üzerinde durarak hataya açık durumları ortadan kaldırmak için çaba gösterir.

Çok iş parçacıklı (Multithreaded): Java'nın çok iş parçacıklı özelliği sayesinde aynı anda birçok görevi gerçekleştirebilen programlar yazmak mümkündür. Bu tasarım özelliği, geliştiricilerin sorunsuz çalışabilen etkileşimli uygulamalar oluşturmasına olanak tanır.

Yorumlanmış (Interpreted): Java bayt kodu anında yerel makine talimatlarına çevrilir ve hiçbir yerde saklanmaz. Bağlantının artımlı ve hafif bir süreç olması nedeniyle geliştirme süreci daha hızlı ve analitiktir.

Yüksek performans (High Performance): Tam Zamanında derleyicilerin kullanımıyla Java, yüksek performans sağlar.

Dağıtılmış (Distributed): Java, internetin dağıtılmış ortamı için tasarlanmıştır.

Dinamik (Dynamic): Java, gelişen ortama uyum sağlayacak şekilde tasarlandığından, C veya C++'dan daha dinamik olarak kabul edilir. Java programları, çalışma zamanındaki nesnelere erişimleri doğrulamak ve çözümlemek için kullanılabilecek geniş miktarda çalışma zamanı bilgisi taşıyabilir.

Java'nın Tarihi

James Gosling, Haziran 1991'de birçok set üstü kutu projesinden birinde kullanılmak üzere Java dili projesini başlattı. Başlangıçta Gosling'in ofisinin önünde bulunan meşe ağacından dolayı 'Meşe' olarak adlandırılan dil, aynı zamanda 'Yeşil' adını da aldı ve daha sonra rastgele kelimelerden oluşan bir listeden Java olarak yeniden adlandırıldı.

Sun, ilk genel uygulamayı 1995 yılında Java 1.0 olarak piyasaya sürdü. Bir Kez Yaz, Her Yerde Çalıştır Write Once, Run Anywhere (WORA) sözü verdi ve popüler platformlarda ücretsiz çalışma süreleri sağladı.

13 Kasım 2006'da Sun, Java'nın çoğunu GNU Genel Kamu Lisansı (GPL) koşulları altında ücretsiz ve açık kaynaklı yazılım olarak yayınladı.

8 Mayıs 2007'de Sun süreci tamamladı ve Sun'ın telif hakkına sahip olmadığı küçük bir kod kısmı dışında Java'nın tüm çekirdek kodlarını ücretsiz ve açık kaynak haline getirdi.

Sırada ne var?

Bir sonraki bölüm Java'yı ve belgelerini nasıl edinebileceğiniz konusunda size rehberlik edecektir. Son olarak Java'nın nasıl kurulacağı ve Java uygulamaları geliştirmek için ortamın nasıl hazırlanacağı konusunda talimatlar verilmektedir.
