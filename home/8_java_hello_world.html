Java - Merhaba Dünya Programı

Çıkış ekranına (konsol) "Merhaba Dünya" yazılması, Java ve diğer programlama dillerindeki ilk programdır. Bu eğitim size Java programlamada ilk programınızı ("Merhaba Dünya" programını yazdır) nasıl yazabileceğinizi öğretecektir.

"Merhaba Dünya" yazdıran Java programı

"Merhaba Dünya" yazısını yazdıracak Java programı aşağıda verilmiştir: 

public class MyFirstJavaProgram {

   /* Bu benim ilk Java programım.
    * Bu, çıktı olarak 'Merhaba, Dünya!' yazacaktır.
    */

   public static void main(String []args) {

      System.out.println("Merhaba, Dünya!"); // Çıktı: Merhaba, Dünya!
   }
}

Merhaba Dünya Programının Açıklaması

Merhaba Dünya'yı çıkış ekranına başarıyla yazdırdığımız için. Kodu satır satır anlayalım.

1. Public Main Class

public class MyFirstJavaProgram {
Bu satır yeni bir MyFirstJavaProgram sınıfı oluşturuyor ve herkese açık olduğundan, bu sınıf MyFirstJavaProgram.java ile aynı isim dosyasında tanımlanmalıdır. Bu kural, Java derleyicisinin dosya içeriğini okumadan önce oluşturulacak genel sınıfın adını belirlemesine yardımcı olur.

2. Comment Section

/* Bu benim ilk Java programım.
* Bu, çıktı olarak 'Merhaba, Dünya!' yazacaktır.
*/
/* */ bloğundaki bu satırlar Java derleyicisi tarafından dikkate alınmaz ve yorumlardır . Yorum, programın daha iyi anlaşılmasına yardımcı olur ve kodun okunabilir ve anlaşılır olmasını sağlar.

3. Public Static Void Main

public static void main(String []args) {
Bu satır, bu program belleğe yüklendiğinde JVM'nin çağırdığı ana yöntemi temsil eder. Bu yöntem programı çalıştırmak için kullanılır. Bu yöntem tamamlandıktan sonra program tek iş parçacıklı ortamda sonlandırılır.

4. Kullanılan Anahtar Kelimeler

Bu satırdaki her anahtar kelimenin amacını kontrol edelim.

public – ana yöntemin kapsamını tanımlar. Herkese açık olduğundan bu yöntem JVM gibi harici programlar tarafından çağrılabilir.

static – ana yöntemin durumunu tanımlar. Statik olan bu yöntem, önce sınıfın nesnesini oluşturmadan JVM gibi harici bir program tarafından çağrılabilir.

void – ana yöntemin dönüş türünü tanımlar. Bu yöntem geçersiz olduğundan herhangi bir değer döndürmez.

main - yöntemin (method) adı

String []args – java komutu yürütülürken komut satırında iletilen argümanlar.

5. System.out.println() Yöntemi (Method)

System.out.println("Merhaba, Dünya!"); // Çıktı: Merhaba, Dünya!
System.out birincil konsolu temsil eder ve println() yöntemi girdi olarak "Merhaba Dünya"yı alır ve aynısını konsol çıktısına yazdırır.
