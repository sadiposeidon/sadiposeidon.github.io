Java - Ortam Kurulumu (Environment Setup)

Java Geliştirme Ortamınızı Kurun

Java programlama dili için kendi ortamınızı oluşturmak istiyorsanız bu eğitim tüm süreç boyunca size rehberlik edecektir. Java ortamınızı kurmak için lütfen aşağıda verilen adımları izleyin.

Java SE ücretsiz olarak indirilebilir. İndirmek için www.oracle.com/in/java/technologies/downloads adresine gidin, lütfen işletim sisteminizle uyumlu bir sürümü indirin.

Java'yı indirmek için talimatları izleyin ve Java'yı makinenize yüklemek için .exe dosyasını çalıştırın. Java'yı makinenize yükledikten sonra, ortam değişkenlerini doğru kurulum dizinlerini işaret edecek şekilde ayarlamanız gerekir.

Windows için Ortam Yolunu Ayarlama

Java'yı c:\Program Files\java\jdk dizinine yüklediğinizi varsayalım.
Windows için Java ortamı yolunu ayarlama adımları aşağıda verilmiştir.

'Bilgisayarım'a sağ tıklayın ve 'Özellikler'i seçin.
'Gelişmiş' sekmesinin altındaki 'Ortam değişkenleri' düğmesini tıklayın.
Şimdi, 'Yol' değişkenini düzenleyin ve sonuna Java yürütülebilir dizininin yolunu ekleyin.
C:\Program Files\java\jdk\bin

Linux, UNIX, Solaris, FreeBSD için Ortam Yolunu Ayarlama

Ortam değişkeni PATH, Java ikili dosyalarının yüklendiği noktaya ayarlanmalıdır. Bunu yaparken sorun yaşıyorsanız kabuk belgelerinize bakın.

Örneğin, kabuğunuz olarak bash kullanıyorsanız, .bashrc dosyanızın sonuna aşağıdaki satırı eklersiniz. PATH'i dışa aktar=/yol/to/java:$PATH'

Popüler Java Editörleri

Java programları yazmak için bir metin düzenleyiciye ihtiyacınız vardır. Piyasada daha da karmaşık IDE'ler mevcut. En popüler olanları aşağıda kısaca açıklanmıştır

Not Defteri – Windows makinesinde Not Defteri (bu eğitim için önerilir) veya WordPad gibi herhangi bir basit metin düzenleyiciyi kullanabilirsiniz. Notepad++ aynı zamanda gelişmiş olanaklara sahip ücretsiz bir metin düzenleyicisidir.

Netbeans – www.netbeans.org/index.html adresinden indirilebilen, açık kaynaklı ve ücretsiz bir Java IDE'sidir.

Eclipse - Aynı zamanda Eclipse açık kaynak topluluğu tarafından geliştirilen bir Java IDE'sidir ve www.eclipse.org adresinden indirilebilir.

IDE veya Entegre Geliştirme Ortamı, kaynak kodu düzenleyicisi, derleme araçları ve hata ayıklayıcılar vb. gibi programlamaya yardımcı olacak tüm ortak araçları ve olanakları sağlar.

Sırada ne var?

Bir sonraki bölümde size ilk Java programınızı nasıl yazıp çalıştıracağınızı ve Java'da uygulama geliştirmek için gereken bazı önemli temel sözdizimlerini öğreteceğim.
