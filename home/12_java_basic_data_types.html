Java - Veri Türleri (Data Types)

Veri türleri, farklı değişken türleri , sabitler, yöntem parametreleri, dönüş türü vb. için verilerin türünü ve değer aralığını tanımlar. Veri türü, derleyiciye depolanacak veri türü ve gerekli bellek hakkında bilgi verir. Farklı veri türlerini depolamak ve işlemek için tüm değişkenlerin belirli veri türlerine sahip olması gerekir.

Java veri türleri iki bölüme ayrılmıştır.

Java İlkel Veri Türleri (Primitive Data Types)

İlkel veri türleri dil tarafından önceden tanımlanır ve bir anahtar kelimeyle adlandırılır. Java tarafından desteklenen sekiz temel veri türü vardır. Aşağıda ilkel veri türlerinin listesi verilmiştir: 
byte
short
int
long
float
double
boolean
char

Java byte Veri Türü

Bayt veri türü, 8 bitlik işaretli ikinin tamamlayıcısı tamsayıdır

Minimum değer -128 (-2^7)

Maksimum değer 127'dir (dahil)(2^7 -1)

Varsayılan değer 0'dır

Bayt veri türü, bir bayt bir tam sayıdan dört kat daha küçük olduğundan, esas olarak tam sayılar yerine büyük dizilerde yerden tasarruf etmek için kullanılır.

Örnek - bayt a = 100, bayt b = -50

Java short Veri Türü

Kısa veri türü, 16 bitlik işaretli ikinin tamamlayıcısı tamsayıdır

Minimum değer -32.768 (-2^15)

Maksimum değer 32.767'dir (dahil) (2^15 -1)

Kısa veri türü, belleği bayt veri türü olarak kaydetmek için de kullanılabilir. Kısa, bir tam sayıdan 2 kat daha küçüktür

Varsayılan değer 0'dır.

Örnek – kısa s = 10000, kısa r = -20000

Java int Veri Türü

Int veri türü, 32 bitlik işaretli ikinin tamamlayıcısı tamsayıdır.

Minimum değer - 2,147,483,648 (-2^31)

Maksimum değer 2,147,483,647(dahil) (2^31 -1)

Tamsayı, bellekle ilgili bir endişe olmadığı sürece genellikle integral değerler için varsayılan veri türü olarak kullanılır.

Varsayılan değer 0'dır

Örnek − int a = 100000, int b = -200000

Java long Veri Türü

Uzun veri türü, 64 bit işaretli ikinin tamamlayıcısı tamsayıdır

Minimum değer -9,223,372,036,854,775,808(-2^63)

Maksimum değer: 9,223,372,036,854,775,807 (dahil)(2^63 -1)

Bu tür int'den daha geniş bir aralığa ihtiyaç duyulduğunda kullanılır

Varsayılan değer 0L'dir

Örnek − uzun a = 100000L, uzun b = -200000L

Java float Veri Türü

Kayan veri türü, tek duyarlıklı 32 bit IEEE 754 kayan noktadır

Float esas olarak büyük kayan nokta sayıları dizilerinde bellekten tasarruf etmek için kullanılır

Varsayılan değer 0,0f'dir

Kayan noktalı veri türü hiçbir zaman para birimi gibi kesin değerler için kullanılmaz

Örnek – float f1 = 234.5f

Java double Veri Türü

double veri türü, çift duyarlıklı 64 bit IEEE 754 kayan noktadır

Bu veri türü genellikle ondalık değerler için varsayılan veri türü olarak kullanılır, genellikle varsayılan seçimdir

Para birimi gibi kesin değerler için asla çift veri türü kullanılmamalıdır.

Varsayılan değer 0,0d'dir

Örnek – çift d1 = 123,4

Java boolean Veri Türü

Boolean veri türü bir bitlik bilgiyi temsil eder

Yalnızca iki olası değer vardır: doğru ve yanlış

Bu veri türü doğru/yanlış koşullarını izleyen basit bayraklar için kullanılır

Varsayılan değer yanlıştır

Örnek – boolean bir = doğru

Java char Veri Türü

char veri türü tek bir 16 bitlik Unicode karakterdir

Minimum değer '\u0000' (veya 0)

Maksimum değer '\uffff' (veya 65.535 dahil)

Char veri türü herhangi bir karakteri saklamak için kullanılır

Örnek – karakter harfiA = 'A'

Örnek: Farklı İlkel Veri Türlerinin Gösterilmesi

Aşağıdaki örnekler yukarıda tartıştığımız çeşitli temel veri türlerinin kullanımını göstermektedir. Boolean ve char değişkenleri bu şekilde yazdırılırken, sayısal veri türlerinde ekleme işlemlerini kullandık. 

public class JavaTester {
   public static void main(String args[]) {

      byte byteValue1 = 2;
      byte byteValue2 = 4;
      byte byteResult = (byte)(byteValue1 + byteValue2);

      System.out.println("Byte: " + byteResult);

      short shortValue1 = 2;
      short shortValue2 = 4;
      short shortResult = (short)(shortValue1 + shortValue2);

      System.out.println("Short: " + shortResult);

      int intValue1 = 2;
      int intValue2 = 4;
      int intResult = intValue1 + intValue2;

      System.out.println("Int: " + intResult);

      long longValue1 = 2L;
      long longValue2 = 4L;
      long longResult = longValue1 + longValue2;

      System.out.println("Long: " + longResult);

      float floatValue1 = 2.0f;
      float floatValue2 = 4.0f;
      float floatResult = floatValue1 + floatValue2;

      System.out.println("Float: " + floatResult);

      double doubleValue1 = 2.0;
      double doubleValue2 = 4.0;
      double doubleResult = doubleValue1 + doubleValue2;

      System.out.println("Double: " + doubleResult);

      boolean booleanValue = true;

      System.out.println("Boolean: " + booleanValue);

      char charValue = 'A';

      System.out.println("Char: " + charValue);     
   }
}


Referans / Nesne Veri Türüleri (Reference / Object Data Types)

Referans veri türleri, sınıfların (classes) tanımlı oluşturucuları (constructors) kullanılarak oluşturulur. Nesnelere erişmek için kullanılırlar. Bu değişkenlerin değiştirilemeyecek belirli bir türde olduğu bildirilir. Örneğin, Employee, Puppy, vb.

Sınıf nesneleri ve çeşitli dizi değişkenleri, referans veri türü kapsamına girer. Herhangi bir referans değişkeninin varsayılan değeri null'dur. Bir referans değişkeni, bildirilen türdeki herhangi bir nesneye veya uyumlu herhangi bir türe başvurmak için kullanılabilir.

Örnek

Aşağıdaki örnek başvuru (demonstrates the reference) (veya nesne (object)) veri türlerini göstermektedir. 

// 'Animal' sınıfının bir nesnesini oluşturmak
Animal animal = new Animal("Zürafa");

// 'String' sınıfının bir nesnesini oluşturmak
String myString = new String("Merhaba, Dünya!");
